# Four-Cylinder Inline Engine CAD Model

## Overview

This project presents a complete four-cylinder inline engine designed and assembled in Autodesk Fusion 360. It demonstrates parametric CAD modeling, mechanical assembly design, engineering visualization, exploded assembly, section views, and motion simulation.

---

## Project Information

| Property | Details |
|----------|---------|
| Software | Autodesk Fusion 360 |
| Project Type | Mechanical CAD Design |
| Engine Type | Four-Cylinder Inline Engine |
| Modeling Method | Parametric CAD Modeling |
| Assembly | Complete Engine Assembly |
| Views Included | Isometric, Front, Left, Right, Top, Section and Exploded Views |
| Motion | Motion Animation |
